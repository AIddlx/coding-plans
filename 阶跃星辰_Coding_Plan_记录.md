## 产品概述

Step Plan是阶跃星辰开放平台专为高频AI开发者打造的订阅制AI服务，针对OpenClaw及AI Coding场景深度定制。让开发者以极具性价比的价格在主流编码工具中使用阶跃星辰旗舰模型。

### 核心特点

- **Prompt-based计费**: 以"prompt"为单位（1 prompt ≈ 15-20次模型调用），透明度高
- **极速默认体验**: 强调"快速响应"为标准配置，不加价
- **用量翻倍优势**: 官方称用量为同档位竞品2倍以上
- **全平台无限制**: 不限制使用平台，一个Plan覆盖主流工具链
- **Agent场景优化**: 专门针对"养龙虾"（OpenClaw Agent）和深度Coding场景优化
- **模型持续扩展**: 当前支持step-3.5-flash，未来将加入更多阶跃旗舰模型

## 用户实测反馈

### Flash Mini 额度使用体验（2026-03-28）

> **用户反馈**: "购买 Flash Mini 套餐后实测，5小时用量和每周用量其实挺多的，体感比其他家的都要多，简单的文本编辑工作根本用不满5小时。"

**对比分析**：
- **标称额度**: Flash Mini 提供 ~100 prompts/5h，按 1 prompt ≈ 15-20 次请求换算，约 1,500-2,000 次请求
- **竞品对比**: 其他家 Lite 档通常为 1,200 次请求/5h
- **实际感受**: 用户反馈实际可用额度明显高于预期，与竞品相比"额度充足感"更强

**原因分析**：
1. **Prompt 归一化**: 阶跃星辰将复杂任务归一化为少量 prompts，避免了因工具调用链过长导致的额度快速消耗
2. **双维度独立**: 5小时额度和周额度独立计算，不均匀使用模式下更灵活
3. **Agent 优化**: 针对 OpenClaw 等 Agent 场景优化，prompt 利用效率更高

**结论**: Flash Mini 的实际可用额度在入门档中具有竞争力，适合中度使用强度的开发者。

### 3. 配置工具

#### Base URL
```
https://api.stepfun.com/step_plan
```
⚠️ **必须使用此专用地址**，普通API地址 `https://api.stepfun.com/v1` 无法使用Step Plan额度！

#### Claude Code示例
```json
{

    "ANTHROPIC_AUTH_TOKEN": "YOUR_API_KEY",
    "ANTHROPIC_BASE_URL": "https://api.stepfun.com/step_plan"
  },
  "model": "step-3.5-flash"

```

### 计费方式

- **Prompt-based**: 与智谱AI一致，透明度高
- **双限额**: 5小时滚动 + 周限额，避免单日过度使用
- **1 prompt ≈ 15-20次调用**: 适合复杂任务场景

### 独特优势

1. **高速默认**: 极速性能为标准配置，不加价
2. **用量翻倍**: 官方宣称用量为竞品2倍+（Mini档≈1,500次 vs 行业平均~600次）
3. **Agent优化**: 专门针对OpenClaw等Agent场景优化，响应快、工具调用效率高
4. **全平台无限制**: 不绑定特定工具，用户自由选择
5. **新模型跟进**: 作为阶跃同名产品，将优先获得最新模型

### 注意事项

- ⚠️ **新发布**: 2026-03-23刚发布，稳定性待验证
- ⚠️ **推广期价格**: 限时优惠，未来可能涨价
- ⚠️ **Max档位量很足**: ¥699/月，适合企业生产环境
- ⚠️ **MCP暂不支持**: MCP工具在规划中，当前版本不支持
- ⚠️ **不限制平台但需合规**: 虽不限制工具，但建议在支持的工具中使用

### 与竞品差异

- **vs 智谱AI**: 都用prompt计费，但Step Fun强调Agent性能，智谱强调MCP工具生态
- **vs 讯飞星火**: 讯飞用tokens计费，Step Fun用prompts；讯飞QPS限制更严格
- **vs 阿里云/百度**: 传统API请求次数计费，Step Fun更透明且额度更高
