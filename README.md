## 🎯 文档内容

每个平台的文档通常包含以下内容：

- **基本信息**：访问时间、官方文档链接
- **产品概述**：套餐定位、核心特点
- **价格与额度**：各档位套餐的定价、prompt/ token 额度
- **配置方法**：API Key 获取、Base URL 设置、工具集成
- **模型支持**：可用模型及切换方法
- **使用限制**：并发、速率、使用场景限制
- **常见问题**：故障排除与使用建议
- **注意事项**：重要提示与避坑指南

## 🔧 支持的编码工具

各平台 Coding Plan 通常支持以下工具（具体以各平台文档为准）：

- **Claude Code** - Anthropic 官方 CLI
- **Cursor** - AI 原生编辑器
- **Open Code** - 开源编码助手
- **Cline / Roo Code** - VS Code 插件
- **Kilo Code** - 轻量级编码工具
- **Cherry Studio** - 桌面应用
- 以及其他 20+ 工具
