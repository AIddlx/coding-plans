## 产品概述

MiniMax Coding Plan 是 MiniMax 官方推出的 AI 编程订阅服务，主打**极速推理**和**多模态能力**。提供标准版和极速版两个系列，满足从入门到重度开发的全场景需求。

### 核心特点

- **高速推理**: 极速版支持 M2.7-highspeed，约 100 TPS，速度达同类产品 3 倍
- **5小时滚动限额**: 采用动态刷新机制，更灵活的使用体验
- **多模态支持**: 支持图像理解、联网搜索 MCP，Plus 及以上支持图像/语音生成，Max 及以上支持音乐/视频生成
- **OpenClaw 集成**: 原生支持 OpenClaw agent，可运行多个并发 agent
- **双速率模式**: 正常约 50 TPS，低峰时段可达 100 TPS（标准版）；极速版恒享 100 TPS
- **灵活订阅**: 支持月付和年付，年付有显著折扣

## 计费模式详解

### 5小时滚动限额

MiniMax Coding Plan 采用 **5小时滚动限额** + **周限额**双重机制：

1. **5小时限额周期**: 每 5 小时刷新一次
2. **周限额**: 周使用额度 = 5小时额度 × 10
3. **额度组成**: 各套餐有固定的"5小时额度"（如 Starter 600次, Plus 1,500次, Max 4,500次）
4. **刷新逻辑**: 每 5 小时结束时，额度重置为满值；周额度每周重置
5. **影响**: 5小时内用尽会阻塞需等待刷新；周额度用尽需等待下周重置

**5小时额度示例**:
- Starter: 每 5 小时最多 600 次调用
- Plus: 每 5 小时最多 1,500 次调用
- Max: 每 5 小时最多 4,500 次调用
- Ultra-极速版: 每 5 小时最多 30,000 次调用

**周额度示例**（5h额度 × 10）:
- Starter: 每周最多 6,000 次调用
- Plus: 每周最多 15,000 次调用
- Max: 每周最多 45,000 次调用
- Plus-极速版: 每周最多 15,000 次调用
- Max-极速版: 每周最多 45,000 次调用
- Ultra-极速版: 每周最多 300,000 次调用

**注意**: MiniMax 无月额度概念，实际月可用量 ≈ 周额度 × 4.3

### TPS 速度说明

| 模型 | 正常 TPS | 低峰 TPS | 极速版 TPS |

| **MiniMax M2.7** | ~50 | ~100（非高峰） | - |

- 极速版**恒享 100 TPS**，不受时段影响，且生成速度是同类产品 3 倍

## 重要限制与风险

**1. 周使用额度限制（2026年3月23日起新用户）：**
- 每周可使用额度为「5小时额度」的 **10倍**
- 例如: Starter (600次/5h) → 周额度 6,000次; Max (4,500次/5h) → 周额度 45,000次
- **老用户保护**: 2026年3月22日23:59:59前购买的用户**不受周额度限制**
- 这意味着: 即使5小时滚动刷新，但如果一周总额度用尽，仍需等待下周重置

**2. 高峰时段动态限流：**
- 限流时段: 工作日 15:00–17:30（根据集群负载动态调整）
- 限流影响: 超高并发自动化任务或多用户共享模式会被限流
- 各套餐支持的持续Agent数量:
  - Starter/Plus: 约支持 1个 Agent 持续调用
  - Max: 约支持 2个 Agent 持续调用
  - Ultra: 约支持 4个 Agent 持续调用
- 限流后通常约1分钟恢复，高峰期可能动态收紧

**3. 生产环境警告：**
- Token Plan **面向个人开发者的交互式使用场景**
- 生产环境建议使用 **pay-as-you-go**（按量付费）方案
- 不适合: 自动化脚本、批量任务、多用户共享服务、自建后端服务

**4. OpenClaw 配置：**
- 如OpenClaw官方版本不支持M2.7，需手动配置:
  1. 编辑 `~/.openclaw/openclaw.json`
  2. 将所有 `M2.5` 替换为 `M2.7`
  3. 重启网关: `openclaw gateway start`
- 默认支持图像理解能力

**5. 额度共享与多工具：**
- 同一订阅可在多个编程工具中同时使用
- 但**所有工具共享同一额度池**
- 建议监控用量，避免多端同时消耗导致快速耗尽

**6. 退款政策：**
- **不支持退款**，无论是否使用完
- 购买前请谨慎选择套餐和周期

**7. API Key 隔离：**
- Token Plan API Key 与普通开放平台API Key **不可互换**
- Token Plan: 按请求次数 + 5小时滚动
- 按量付费: 按实际token消耗，无额度限制

**8. 升级灵活性：**
- 支持订阅期内随时升级（包括标准版→极速版）
- 只需支付差价，新套餐立即生效

**9. ⚠️ 极速版性价比严重偏低（重要）：**
- **Plus-极速版（¥98）** vs 标准版Plus（¥49）：价格**翻倍**，但5h额度完全相同（1,500次），每元额度从30.6降至15.3，**性价比腰斩**
- **Max-极速版（¥199）** vs 标准版Max（¥119）：价格上涨**67%**，额度相同，每元额度从37.8降至22.6，**性价比下降40%**
- **Ultra-极速版（¥899）**：无标准版对比，但每元额度33.4次远低于阶跃Pro（227次/元）和Flash Plus（122次/元）
- **核心提示**：极速版的溢价只为"恒享100 TPS"（不受工作日15:00-17:30高峰限流）。标准版在低峰时段也可达100 TPS，仅高峰时段可能受限。**除非工作流严格依赖高峰时段且已验证限流严重影响效率，否则标准版性价比显著更高**。
