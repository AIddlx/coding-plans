## 产品概述

优云智算Coding Plan是优刻得旗下的大模型API订阅服务，提供积分制计费模式，支持DeepSeek、MiniMax、Kimi、GLM-5等全球16个顶尖大模型。通过统一API接口，兼容OpenAI和Anthropic协议，可接入Claude Code、OpenClaw、Cursor等主流AI编程工具。

### 核心特点

- **积分制灵活计费**: 不同于Token或API请求次数，采用多维度积分消耗模型
- **价格实惠**: 远低于官方API价格，适合个人开发者
- **不限制场景**: 既可用于编程工具，也支持通用API调用
- **多模型覆盖**: 整合多个厂商的SOTA模型，统一接入
- **可叠加套餐**: 按量包与包月套餐可叠加使用，灵活管理
- **使用透明**: 提供详细统计和调用日志

## 计费模式详解

### 积分消耗公式

```
总积分消耗 = 
  输入Token × 输入倍率 +
  缓存创建Token × 缓存创建倍率 +
  缓存命中Token × 缓存命中倍率 +
  输出Token × 输出倍率
```

**关键概念**:
- **输入倍率**: 不同模型不同（通常1-3倍）
- **缓存创建/命中倍率**: 利用prompt缓存可大幅节省积分
- **输出倍率**: 通常最高（如2-4倍）

> 💡 **提示**: 具体模型倍率请查看控制台"套餐购规则页面"，不同模型差异较大。

### 成本估算

以畅享包 Lite（¥49.9/月，700万积分/日）为例：
- 假设平均每次编码对话消耗10,000积分
- 每日可进行约 **700次** 编码对话
- 单次对话成本约 **¥0.007**（0.7分钱）

相比按量包：
- 体验包¥6.9得2,900万积分 → 约2,900次对话（假设1万积分/次）
- 单次成本约 **¥0.0024**（0.24分钱）

## 快速开始

### 1. 购买套餐
- 访问[优云智算Coding Plan页面](https://www.compshare.cn/coding-plan)
- 选择按量包或包月套餐
- 完成支付

### 2. 获取API Key
- 进入[控制台](https://console.compshare.cn/light-gpu/api-keys)
- 创建API Key（选择Coding Plan类型）
- 复制保存

### 3. 配置工具

**Base URL**:
- OpenAI协议: `https://api.modelverse.cn/v1`
- Anthropic协议: `https://api.modelverse.cn`

**模型ID**: 使用 `modelverse-<model-name>` 格式，如 `modelverse-deepseek-v3.2`（具体模型名称见控制台）

#### Claude Code配置示例
```json
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "你的API Key",
    "ANTHROPIC_BASE_URL": "https://api.modelverse.cn",
    "ANTHROPIC_MODEL": "modelverse-deepseek-v3.2"
  }
}
```

#### OpenClaw配置示例
```json
{
  "models": {
    "providers": {
      "modelverse": {
        "baseUrl": "https://api.modelverse.cn/v1",
        "apiKey": "你的API Key",
        "models": [
          { "id": "modelverse-kimi-k2.5", "name": "Kimi K2.5" }
        ]
      }
    }
  }
}
```

### 4. 开始使用
- 启动工具（Claude Code、OpenClaw等）
- 查看[控制台-套餐统计](https://console.compshare.cn/light-gpu/statistics)监控使用情况
